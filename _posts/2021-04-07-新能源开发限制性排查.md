本说明文档用于指导风电、光伏项目在**前期选址与方案比选阶段**，基于已有空间数据，系统排查各类限制性因素，形成：

- 禁止建设区
- 严格受限区 / 条件建设区
- 优先建设区

适用于：

- 集中式风电场 / 光伏电站选址
- 分散式（分布式）风电项目布点


## 数据类型与排查流程概览

限制性因素主要分为两类数据：

- **矢量数据（Vector）**：点 / 线 / 面约束，适合做缓冲区、叠加分析；
- **栅格数据（Raster）**：地类、坡度、坡向等连续空间信息，适合阈值筛选、重分类。

一个典型的排查流程如下：

1. **数据准备与统一坐标**

   - 统一为同一投影坐标系（如 EPSG:3857 / 4490 等），检查精度与缺失。

2. **矢量限制叠加与缓冲**

   - 将已建项目、军事区、保护区、道路、线路等叠加到底图，按规范设置缓冲（如道路两侧一定距离内不布置风机/组件）。

3. **栅格限制筛选**

   - 对地类、坡度、坡向等栅格数据按阈值进行重分类，生成“可建设 / 不可建设”二值掩膜。

4. **综合叠加与分级**

   - 将矢量与栅格掩膜统一到同一分辨率 / 分区，形成最终的限制性分区图层，为后续资源评估、经济性分析提供边界条件。


## 通用限制性因素（风 / 光共用）

### 矢量限制因素

下列矢量要素在风电、光伏项目中通常视为**禁止或严格限制建设区域**，需在前期即予以剔除或高权重约束：

- **已建风电 / 光伏电厂**

  - 避免与既有场站用地、道路、集电线路等冲突。

- **军事区域**

  - 各类军用基地、靶场、雷达站等，通常为严格禁入区，并有高度与电磁方面的额外要求。

- **地质灾害点**

  - 地灾隐患点、滑坡、泥石流等，周边一定范围内不宜新建风机基础、箱变或组件支架。

- **矿区与采空区**

  - 采矿活动区域、采空塌陷区，需避让或进行专项稳定性论证。

- **自然保护区及各类生态红线**

  - 自然保护区、国家公园、自然遗产地、湿地公园、地质公园、风景名胜区等，通常为**风电/光伏建设禁入区**。

- **水体**

  - 湖泊、水库、河道及其管理范围。除专门规划的漂浮式光伏等特殊项目外，一般视为不可建设。

- **道路**

  - 高速公路、国省道、县道等，需考虑安全距离、眩光影响以及施工交通条件。

- **输电线路 / 通信线路**

  - 高压输电线、通信线等，需要控制安全净距，避免风机叶片、塔架或光伏支架进入安全距离范围。

### 栅格限制因素

常用于限制性分析的栅格数据包括：

- **地类 / 土地利用类型**

  - 地类 160 / 170 / 180：灌木或草本覆盖、周期性淹没、淡水 / 咸水 / 微咸水等区域，多数情况下建设条件较差或生态敏感。
  - 地类 190：建成区（城镇建设用地），一般不考虑集中式风光电站布置。
  - 地类 210：水体。
  - 地类 220：永久冰雪区。

  > 以上地类通常整体作为**排除区域**或高成本区域处理。

- **坡度（Slope）**

  - 通常设定一个坡度阈值，例如：

    - 坡度 ≤ 30°：可考虑建设（其中 0–15° 较为适宜，15–30° 条件较差但可用）；
    - 坡度 > 30°：视为不适宜建设区，主要出于土石方工程量、安全与运维成本考虑。

- **雪线**

  - 雪线以上区域通常不纳入常规风光项目选址，除非是专门的高寒高海拔方案。


## 风电开发的限制性因素

### 机场及飞行安全

风电场与机场、航线的关系，需要重点核查：

- 项目是否位于民航或军用机场一定半径范围内；
- 风机总高度（塔筒 + 叶片）是否对飞行安全、雷达探测造成影响；
- 是否处在净空保护区、障碍物限制面范围内。

这部分通常需要结合民航、军方等部门的专项审查意见，在空间分析中先进行**硬性排除**，再根据批复细化边界。

### 林地与生态敏感区（风电特有关注）

根据现行林草管理要求，以下区域原则上不得新建风电场或需从严控制：

- 自然保护区、国家公园、自然遗产地等高等级保护区域；
- 鸟类主要迁徙通道和停歇地；
- 沿海基干林带、消浪林带等重要防护林；
- 天然乔木林（竹林）、部分国家级公益林、低降雨区有林地等。

在实际排查中，可将上述区域统一归类为**禁止建设区**，并对风机基础、升压站、施工道路、集电线路等建设内容分别控制用地边界。

### 地形复杂区的风电选址要点（简要）

在复杂山地地形中，风电选址除了“能不能建”，还要考虑“风能资源是否被地形放大或削弱”。可在限制性分析中加入以下经验性约束：

- **山脊相对主导风向的方位**：优先考虑山脊大致垂直于主导风向的地形。
- **坡面连续性**：从山前到山脊的上升坡度应相对连续、不过于破碎。
- **山尖形态**：过于平缓的山顶可能风速增益有限。
- **坡度与高度的综合**：陡坡虽有一定风速增益，但随高度增加加速效应衰减较快，需要结合机位高度综合判断。

这些内容可以不作为“硬性禁止条件”，但建议在 GIS 分析结果上叠加风资源模拟结果，共同指导机位布设。


## 光伏开发的限制性因素

相较风电，**地形对光伏的直接影响更大**，特别是坡度与坡向：

### 坡度

在常规地面电站场景下，可参考如下分级（可根据项目自由调整阈值）：

- **0–10°：近似平地**

  - 场区平整成本较低，施工、运维条件好，优先推荐。

- **10–15°：基本适宜**

  - 需要一定的平整及支架调节，但整体可控。

- **15–30°：条件可用但成本升高**

  - 土石方工程量增加，支架形式与排布复杂度提高，应结合经济性评估。

- **>30°：原则上不考虑**

  - 建设与运维风险显著放大，一般在前期筛选中直接排除。

> 在实际建模时，可以将不同坡度等级赋予不同“成本权重”，而不仅仅是简单二分为“可建 / 不可建”。

### 坡向

坡向直接影响光照利用效率，可在栅格筛选中增加坡向条件：

- **南向坡（北半球）：优先**

  - 接收太阳辐射较多，可作为光伏优选区。

- **东 / 西向坡：次优**

  - 早晚照较偏，整体发电量较南向略低，但在地形受限区仍可接受。

- **北向坡：一般不考虑**

  - 如项目资源紧张或有特殊需求，可通过专项模拟评估。

### 雪线及极端气候区

雪线以上及极端寒冷区域，在光伏组件覆冰、运维通行等方面有明显不利因素。
这类区域通常在前期作为**排除区**处理，或在专项评估后单独建模。


## 分散式风电项目的选址避让原则

分散式风电项目通常更靠近负荷中心或乡镇、村庄分布区域，对**用地类型与周边敏感点**的要求更高，选址时建议重点避开：

- 基本农田、优质耕地；
- 各类林地，尤其是公益林、天然林；
- 矿产开发区、规划矿权范围；
- 各级自然保护区、风景名胜区、地质公园等片状敏感区域；
- 军事设施、文物保护单位、历史遗址等点状敏感目标；
- 居民集中区、学校、医院等对噪声、景观敏感的建筑群。

在 GIS 层面，可以通过：

- 为片状敏感区设置“完全禁止”掩膜；
- 为点状敏感区（如文物点、村庄点）设置一定半径的缓冲区；
- 在最终可建设区内再进行机位或线路细化布设。
