本站

建立个人网站一方面是为了分享知识，我在工作和生活过程中接触到大量优秀的文章教程和开源软件，今天互联网技术的繁荣背后来自这些人的无私奉献。 我愿尽自己的绵薄之力，把好的内容分享给更多人，并在这个过程中记录一些自己的收获体会。

另一方面也是为了锻炼写作能力，因为写作是最好的思考方式。对一个概念是否正真理解，从落笔的那一刻起便直观的向读者呈现出来。 天才物理学家费曼先生提出过一种学习技巧：将专业的知识通过自己的语言表达，以一种外行人也能听懂的方式把这件事讲清楚，通过这种方式来达到正真的理解 —— 这是也我试图通过写作、通过知识分享、通过建立个人网站努力达到的目标。

本站使用了 Jekyll 作为框架， 在 jekyll-theme-yat 主题的基础上依据个人审美做出了一些微小的调整。 项目部署在 GitHub Page 上，源代码参见 GitHub 仓库。
